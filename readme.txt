五子棋练习程序


======== 功能特点 ========
约1400行代码。实现了完整的五子棋游戏单机版功能，具有以下功能特点：

1、人工智能。提供了两个风格不同的电脑AI。
2、可定义任意大小的棋盘。界面和AI能自适应棋盘大小。
3、可以悔棋(任意步)、自动代下、电脑托管。
4、可以复盘，重头开始观看棋局。并且从任意一步的历史状态开始恢复为对弈状态。
5、可以保存对局。下次重新打开继续下，或者复盘演练。
6、平局检测。当确定盘面上任意一方都无法凑成五子后，自动结束棋局为平局。
7、高性能，后台模拟AI对局在2~3ms左右即可完成一局。可以用大量AI对战通过胜率计算来分析AI的好坏。
8、仅凭Java环境即可运行，不依赖任何第三方库。

======== 游戏说明 ========

游戏界面具有以下功能
“游戏”菜单
1、可以开始游戏。黑子先下，白子后下，用户可选择执黑或执白。自动模式为电脑对下。
2、重新定义棋盘大小。可以输入棋盘大小，格式为“长x宽”。重新定义棋盘后原先的对局不能保留，只能开始新游戏。

“自动”菜单
1、自动下一子。即由电脑代下一步。
2、电脑托管，将当前棋手改由电脑代下。

“历史”菜单
1、可以悔棋
2、保存棋局可以将当前棋局包含历史步骤等全部信息保存下来。
3、载入棋局，可以载入保存的棋局接着玩。

“复盘”菜单
1、进入复盘模式，即从第一步开始，演示之前的步骤。可以从菜单选择“前进”或“后退”。也可在屏幕上按鼠标左键是前进一步，按鼠标右键后退一步。
2、“退出复盘模式”，即跳回到复盘前的棋局上。
3、“从当前局面开始下棋”，即在一个历史点上重新开始对弈。


======== AI扩展和后台自动AI对战 =========
可以自行编写类，实现AI接口。在ai.properties中配置类名，即可用指定的AI替换白棋或黑棋的电脑智能。
默认提供了Default AI和Active AI两种，后者在进攻上更积极。

评估AI的好坏，可以用让AI自行对战，计算胜率的方法来进行。
Chess类提供了main方法，执行java Chess即可用最快速度在后台执行1000次电脑自动对局，并统计胜率。
输出胜率之外，还可以统计耗时。

======== 命令行方式交互 =========
在linux或unix等命令行方式下,可以使用命令行方式游戏(需要GBK字符支持。)
运行
java ChessConsole
即可进入命令行方式。通过输入命令，即可交互下棋。

命令列表如下。
new          开始新棋局
chess m,n    将棋盘定义为xn大小。
state        显示棋盘    [可省略为st]
cmove        自动下一子，[可省略为c]
comp         电脑托管
roll         悔棋        [可省略为r]
review       复盘 (在复盘模式下，电脑落子即为前进, roll命令即为后退)
resume       （在复盘中）退出复盘
play         （在复盘中）从以当前棋局开始下棋。
m,n           (输入两个数字，逗号分隔)，在指定位置上下子。
save 文件名   保存棋局
load 文件名   载入棋局
exit         退出
